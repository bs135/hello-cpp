# hello-cpp

This is the hello-cpp project.

## Building and installing

For first time

```bash
rm -rf build
cmake -S . -B build
```

build using cmake command

```bash
cmake --build build

# run
./build/hello-cpp

# clean
cmake --build build --target clean

```

```bash
cd build
make

# run
./hello-cpp

# clean
make clean
```
