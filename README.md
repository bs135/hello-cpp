# hello-cpp

This is the hello-cpp project.

## install dependencies

```bash
sudo apt install -y \
    build-essential \
    libssl-dev \
    cmake \
    nlohmann-json-dev \
    libspdlog-dev

```

## Building and installing

```bash
make build
cd build
cmake ..
make

# run
./hello-cpp

# clean
make clean
```
